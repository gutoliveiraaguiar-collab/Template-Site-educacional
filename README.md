# 🏫 Portal Escolar Template

Template de site institucional para escolas, desenvolvido com HTML5 e CSS3 puro.

## 🎨 Funcionalidades

### 📱 Design Responsivo
- Desktop: layout em grid (2 colunas)
- Tablet: layout adaptado
- Mobile: layout em coluna única

### 🧩 Componentes inclusos
- ✅ Header com logo e menu responsivo
- ✅ Banner principal
- ✅ Cards de notícias
- ✅ Seção de cursos
- ✅ Área de avisos laterais
- ✅ Footer com contatos e redes sociais

### 🎯 Tecnologias utilizadas
- **HTML5** (estrutura semântica)
- **CSS3** (estilização)
- **Flexbox** (menu e cards)
- **Grid Layout** (layout principal)
- **Media Queries** (responsividade)
